# Agentic AI Learning Repository

This repository contains my learning journey into Agentic AI, including different agent frameworks, projects, and course-based exercises.

## What is Agentic AI?
Agentic AI refers to systems where AI models act as **autonomous agents** that can:
- Plan tasks
- Use tools
- Communicate with other agents
- Solve complex problems step by step
